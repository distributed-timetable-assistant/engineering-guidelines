# زبان‌های برنامه‌نویسی

به بخش زبان‌های دستورالعمل‌های مهندسی Dita خوش آمدید. در اینجا استانداردها، بهترین روش‌ها و الزامات ابزار برای زبان‌های برنامه‌نویسی که استفاده می‌کنیم را خواهید یافت.

## زبان‌های پشتیبانی شده

*   [Rust](./rust/README.md) - دستورالعمل‌هایی برای نوشتن سیستم‌های ایمن، همزمان و کاربردی در Rust.

**اهداف این بخش:**
*   تضمین یکپارچگی در تمام کدبیس‌ها.
*   ترویج روش‌های اصولی مختص هر زبان.
*   تعریف ابزارها و پیکربندی استاندارد (لینترها، فرمت‌کننده‌ها).
